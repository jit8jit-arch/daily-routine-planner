# DayFlow — Daily Routine & Reminder

A mobile-first PWA for daily routine timetable, medicine reminders, and supplement tracking. Designed for Android and iOS. Deploys free on GitHub Pages.

---

## 🚀 Deploy to GitHub Pages (3 steps)

### Step 1 — Create the repo
1. Go to [github.com/new](https://github.com/new)
2. Name it `dayflow` (or any name you like)
3. Set visibility to **Public**
4. Click **Create repository**

### Step 2 — Upload files
Upload these 3 files to the repo root:
- `index.html`
- `manifest.json`
- `sw.js`

_(Drag & drop them in the GitHub web interface)_

### Step 3 — Enable GitHub Pages
1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / root
4. Click **Save**

Your app will be live at:
**`https://YOUR_USERNAME.github.io/dayflow/`**

---

## 📱 Install on Phone

### Android (Chrome)
1. Open the URL in Chrome
2. Tap the **⋮ menu → Add to Home screen**
3. Tap **Add** — it installs like a native app

### iPhone (Safari)
1. Open the URL in Safari
2. Tap the **Share icon (□↑)**
3. Tap **Add to Home Screen**
4. Tap **Add**

## ✨ Features

| Feature | Detail |
|---|---|
| 📋 Routine Timetable | Morning / Afternoon / Evening / Night sections |
| 💊 Medicine Reminders | Multiple daily dose times per medicine |
| 🧴 Supplement Tracking | Separate supplement tab with dose info |
| ✅ Progress Ring | Visual day-completion tracker |
| 🔔 Browser Notifications | Optional push reminders |
| 📴 Works Offline | Service worker caches everything |
| 🌙 Dark Theme | OLED-friendly dark UI |
| 💾 Local Storage | All data persists in your browser |
| 🔄 Repeat Rules | Daily / Weekdays / Weekends / Once |
| 🎨 Colour Coding | 6 colour options per task |

---

## 🔧 Customization

All data is stored in your browser's `localStorage` — no server, no login, fully private.

To reset to defaults, go to **Settings → Clear All Data**.
